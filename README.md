# CodeAlpha_StudentManagementSystem

This is a simple program written in C to manage student records.

What it can do:

* Add a student
* Show all students
* Search for a student
* Delete a student
* Update a student


How to use:

When you run the program, you will see a menu.

Type a number:

* 1 → add student
* 2 → show students
* 3 → search
* 4 → delete
* 5 → update
* 6 → exit

## Notes

* Data is saved in a file, so it stays even after closing the program
* If no data, it will show "empty"
